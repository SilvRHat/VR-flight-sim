DEMO Project

DEMO - Proof of Concept

 Easy - 2 levels - Boeing T45C

This demo showcases the following technologies for building guided runs in Prepar3D:

  • Reading/Writing scenario variables
  • Direct user interfaces using camera manipulation
  • Surveying at specific trigger points (gates)
  • Collecting survey data
  • Stringing scenarios into a series
  • User feedback on input (sound/ FX)
  • Guiding user path towards gates
